# Rulers

Simple web framework

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rulers'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rulers

## Usage

Include the gem and start using it in your own app.
